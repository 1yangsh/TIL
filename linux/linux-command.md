# 리눅스 명령어

> 리눅스에서 사용하는 기본 명령어 정리. 
>
> 윈도우 기준 => git bash 프로그램 사용



## 리눅스 시스템

- `.` : 현재폴더
- `..` : 상위폴더
- 앞에 `.`이 붙은 파일, 폴더 => 숨겨져 있음



### pwd

- *print working directory*의 약자
- 현재 나의 경로(위치)를 출력해주는 명렁어



### cd

- change directory의 약자
- 디렉토리 이동을 위해 사용하는 명령어
  - `cd <이동하고 싶은 폴더의 이름>`
  - `cd ~`  // 기본 경로로 이동
  - `cd ..`  // 상위 폴더로 이동



### ls

- list의 약자로 현재 디렉토리에 있는 파일과 폴더를 출력
- `-a`옵션
  - 숨김파일, 숨김폴더까지 모두 출력



### mkdir

- 디렉토리(폴더)를 생성하는 명령어
- `mkdir <폴더명.확장자>`



### touch

- 파일을 생성하는 명령어

- `touch <파일명.확장자>`
