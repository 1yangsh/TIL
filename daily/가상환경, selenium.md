#### 가상환경 만들기

> 버전관리 등을 위해



- 현재 환경 정보

  - 아나콘다 프롬프트에서 (관리자 권한 실행)  `conda info --envs` 

    

- 가상환경 만들기

  - `conda create --name <이름> (파이썬 버전)`

    

- 가상환경 실행

  - `conda activate <이름>`

    

- 가상환경 종료

  -  `conda deactivate`

    

- 가상환경 삭제

  - `conda remove --name <이름> --all`



---



- `conda list`

  - 현재 라이브러리 목록을 보여줌

    

- `conda install selenium`   /   `pip install selenium`

  - selenium 설치
