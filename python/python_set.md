## set (집합)

- 중복 허용 안함

```python
# set 예제
s = set([1,2,3,1,2,3])  # 결과 값 {1,2,3}
s = remove(1)           # 결과 값 {2,3}
s.update([1,4,5,6,7])   # 결과 값 {1,2,3,4,5,6,7}
s.discard(3)			# 결과 값 {1,2,4,5,6,7}
s.clear()				# 결과 값 없음

```

```python
s1 = set([1,2,3,4,5])
s2 = set([3,4,5,6,7])

s1.union(s2)            # 결과 값 {1,2,3,4,5,6,7} = s1 | s2
s1.intersection(s2)     # 결과 값 {3,4,5}         = s1 & s2
s1.difference(s2)       # 결과 값 {1,2}		   = s1 - s2
```
